LOBSTER
=======

**LOBSTER** provides traceability extraction, aggregation and the
``lobster-ci-report`` gate that enforces full up/down traceability across
use cases, tool requirements, source code, and tests.

LOBSTER is consumed as an external Bazel module. For full documentation see
`LOBSTER on GitHub <https://github.com/bmw-software-engineering/lobster>`_.
