TRLC
====

**TRLC** (Tool Requirements Language for C) is the requirements DSL used to
write use cases, potential errors, and tool requirements. The ``trlc_rst``
rule renders ``.trlc`` files into Sphinx-compatible RST for inclusion in
this documentation.

TRLC is consumed as an external Bazel module. For full documentation see
`TRLC on GitHub <https://github.com/bmw-software-engineering/trlc>`_.
