# SCORE Architecture Levels

Project-specific architecture levels for the SCORE Architecture Design Process.
These complement the general architecture guidelines.

| Level | Scope | Diagram Kinds |
|---|---|---|
| **Feature Architecture** | Integration-level structure and interactions | Component, sequence |
| **Component Architecture** | Component-internal structure | Class, component |
| **Public API** | Externally visible interfaces of a component | Component (public_api) |
